# Historical docs

Pre-launch planning artifacts. **Do not use these as the source of truth** — they
predate the 2026-05-15 VC consensus loop that collapsed the pricing model from
four tiers (Starter / Growth / Scale / Accountant Pro) to two (Business $79 /
Accountant $299) and re-framed the AI from "auto-discovery" to "suggestion
engine."

Kept for paper trail, not for guidance:

- `OperatorOS_Project_Plan.md` — the long-form planning bible. Still has
  intellectual scaffolding for unit economics, GTM, and roadmap framing, but
  every concrete number (pricing, tier features, AI scope) is the v1 model.
- `OVERVIEW.md` — investor-facing overview, v1 pricing.
- `PITCH.md` — sales pitch, v1 pricing.
